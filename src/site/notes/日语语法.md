# 前言
---
什么是语法：语言的规律。

各种各样的日语语法体系：山田语法、桥本语法、时枝语法、学校语法、教育语法等等。

日语语法总体可以分为三大部分，分别是词法、句法和篇章法。

日语语法结构：[[日语语法结构图.canvas\|日语语法结构图]]

# 词法
---
日语中的各种词从有无形态变化角度可以分为2大类。一类是有形态变化的“活用词”，一类是没有形态变化的“无活用词”。

日语中的活用词有以下几类。
- [[【动词】\|【动词】]]
- [[【形容词】\|【形容词】]]
- [[【助动词】\|【助动词】]]

日语中的无活用词有以下几类。
- [[【名词】\|【名词】]]
- [[【代词】\|【代词】]]
- [[【数词】\|【数词】]]
- [[【副词】\|【副词】]]
- [[【接续词】\|【接续词】]]
- [[【助词】\|【助词】]]
